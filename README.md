基于 Godot 4.6 引擎的像素风 **类吸血鬼幸存者** 射击游戏。主角是人类幸存者，使用 WASD 移动、自动射击僵尸、升级成长，在尸潮中尽可能活下去。

## 游戏玩法

### 核心循环

1. **主菜单** → 点击"开始游戏"
2. **选关地图** → 自由行走接近关卡圆点 → 点击"进入游戏"
3. **战斗** → 自动射击僵尸，拾取经验球升级
4. **升级** → 选择强化能力（每次三选一）
5. **死亡** → 查看生存时间和得分 → 返回选关地图

### 战斗机制

- **自动射击**：主角自动朝**朝向方向**射击最近的僵尸
- **僵尸撕咬**：僵尸靠近后会**咬住**玩家，持续造成伤害
- **挣脱**：移动拉开距离可挣脱僵尸
- **升级成长**：击杀僵尸获得经验，升级时三选一强化

## 操作说明

| 按键 | 选关地图 | 战斗中 |
|------|----------|--------|
| W / ↑ | 向上行走 / 切到上一章 | 向上移动 |
| S / ↓ | 向下行走 / 切到下一章 | 向下移动 |
| A / ← | 向左行走 / 向左切关 | 向左移动 |
| D / → | 向右行走 / 向右切关 | 向右移动 |
| 鼠标点击 | - | 无（自动射击） |

## 选关地图

5 个章节，每章 4-5 个关卡：

| 章节 | 关卡数 | 颜色说明 |
|------|--------|----------|
| 第1章 | 5 关 | 🔵 蓝色=已解锁 |
| 第2章 | 4 关 | 🟢 绿色=已通关 |
| 第3章 | 5 关 | ⚫ 灰色=未解锁 |
| 第4章 | 4 关 | 🟡 金色光圈=选中 |
| 第5章 | 5 关 | |

- 玩家角色（蓝色小人）在地图上自由行走
- 走到已解锁的蓝色圆点附近 → 自动选中 → 底部显示关卡信息 + "进入游戏"按钮
- 关卡之间用线连接，形成可视化进度

## 角色与敌人

### 玩家

- **外观**：16×16 像素人类幸存者（蓝色上衣、持枪）
- **属性**：生命值 100、移速 280、自动回复、自动射击
- **朝向**：左右转向，射击优先瞄准朝向侧的敌人

### 僵尸

| 类型 | 生命 | 速度 | 撕咬伤害 | 经验 |
|------|------|------|----------|------|
| 🟩 普通僵尸 | 30 | 100 | 6/0.6s | 5 |
| 🟧 快速僵尸 | 15 | 180 | 4/0.6s | 3 |

僵尸行为：
1. **追踪** → 朝玩家移动
2. **咬住** → 靠近 18px 后咬住玩家，持续掉血
3. **挣脱** → 拉开 45px 距离后松口

## 升级系统

击杀僵尸获得经验球，拾取累积经验值升级。
升级公式：`所需经验 = 10 × 1.25^(等级-1)`

每次升级从以下 10 种能力中随机 3 个供选择：

| 升级 | 效果 | 最大等级 |
|------|------|----------|
| ❤️ 护甲强化 | +20 最大生命 | 5 |
| 💚 纳米修复 | +0.3 HP/秒 回复 | 5 |
| 🚀 加力燃烧 | +30 移速 | 5 |
| 💥 穿甲弹 | +20% 子弹伤害 | 5 |
| 🔫 自动装弹 | +15% 射速 | 5 |
| 🎯 枪口增推 | +20% 子弹速度 | 5 |
| 📏 重型弹头 | +15% 子弹尺寸 | 5 |
| 🔗 双联装 | +1 额外弹道 | 3 |
| 🧲 牵引光束 | +40 拾取范围 | 5 |
| 📚 战场研究 | +20% 经验获取 | 5 |

## 视觉效果

- **屏幕震动**：受伤时摄像机抖动
- **暂停帧**：升级时短暂慢放
- **粒子爆散**：僵尸死亡（橙色）、经验拾取（绿色）、受伤（黄色）
- **星空背景**：200 颗星星缓慢滚动
- **无敌闪烁**：受伤后 0.5 秒半透明闪烁

## 技术细节

- **引擎**：Godot 4.6（GL Compatibility 渲染模式）
- **语言**：GDScript 2.0
- **纹理**：运行时用 `Image.create()` 生成像素画纹理，无需外部素材
- **输入**：通过 `Input.get_vector()` / `_unhandled_key_input()` 处理
- **碰撞层**：玩家=2，敌人=4，子弹碰撞敌人

### 自动加载 (Autoload)

| 名称 | 作用 |
|------|------|
| `AssetDB` | 像素纹理仓库 |
| `GameManager` | 游戏状态、经验、分数 |
| `UpgradeManager` | 升级池和选择 |
| `ProgressManager` | 关卡进度追踪 |
