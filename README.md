## 游戏玩法

### 核心循环

```
主菜单 → 选关地图 → 进入关卡
                        ↓
              三波尸潮（15→20→25只）
                        ↓
             击杀僵尸得经验、掉金币
                        ↓
           升级（拾取进背包 或 直接装备）
                        ↓
           三波全清 → 过关 → 下一关
```

### 战斗机制

- **自动射击**：自动瞄准最近的僵尸
- **僵尸撕咬**：僵尸靠近后咬住玩家，持续掉血，拉开距离挣脱
- **击退效果**：子弹命中僵尸会将其弹开，打断撕咬，轻量僵尸弹得更远
- **金币掉落**：僵尸死后掉金币，永久留在地上直到拾取
- **经验直得**：击杀立即获得经验，无需拾取

## 波次系统

每关 **3 波僵尸**，底部三条进度条并排显示：

```
████████████   █████░░░░░░   ░░░░░░░░░░
   15/15          8/20          0/25    （击杀/目标）
  🟢已完成        🟠当前波        ⚫未开始
```

| 波次 | 击杀目标 | 快速僵尸概率 | 一击杀数量 |
|------|:--:|:--:|:--:|
| 第1波 | 15 | 0% | 1 |
| 第2波 | 20 | 20% | 1-2 |
| 第3波 | 25 | 35% | 2 |

- 波次间隙 **2 秒**暂停生成
- 三波全清 → **关卡通过** → 弹出面板（下一关 / 返回地图）

## 背包系统

在战斗中按 HUD 右上角 **「背包」** 按钮打开：

| 属性 | 值 |
|------|----|
| 布局 | 5 × 6 方形格子 |
| 总格子 | 30 |
| 默认解锁 | 10 格 |
| 锁定格子 | 灰色 + 🔒，20 格 |
| 解锁价格 | 50 金币 / 格 |
| 使用方式 | 升级时选「拾取」→ 存入背包 → 打开背包点「装备」生效 |

## 角色与敌人

### 玩家属性

| 属性 | 初始值 | 说明 |
|------|:--:|------|
| 生命值 | 100 | 归零游戏结束 |
| 恢复 | 0.5/秒 | 自动回血 |
| 移速 | 280 | 像素/秒 |
| 体型 | 1.0 | 影响被击退程度 |

### 僵尸类型

| 类型 | 外观 | HP | 速度 | 咬伤 | 体型 | 经验 | 出现 |
|------|------|:--:|:--:|:--:|:--:|:--:|------|
| 🧟 普通僵尸 | 16×16 绿棕色，红眼 | 30 | 100 | 6 | 1.0 | 5 | 全局 |
| 🏃 快速僵尸 | 12×12 橙褐色，红眼 | **8** | 180 | 4 | **0.5** | 3 | 第2波起 |

僵尸行为：追踪 → 靠近咬住 → 被击退打断 → 继续追踪

## 击退系统

| 参数 | 公式 / 值 |
|------|------|
| 击退力 | `子弹伤害 × 3.0` |
| 实际速度 | `击退力 ÷ max(体型, 0.3)` |
| 硬直时间 | 0.12 秒（期间不追击） |
| 衰减速度 | 30/秒 |
| 打断撕咬 | ✅ |

- 普通僵尸（体型 1.0）→ 弹一个身位
- 快速僵尸（体型 0.5）→ 弹两个身位，明显推飞

---

## 金币系统

- **掉落**：每只僵尸死亡掉落 1 个金币实体（黄色圆点）
- **存放**：永久留在地上，不会消失
- **拾取**：靠近后自动吸附，触碰增加金币
- **显示**：HUD 右上角黄色字体实时显示
- **用途**：在背包中 50 金币解锁一个格子

---

## 升级系统

经验公式：`所需经验 = 10 × 1.25^(等级-1)`

每次升级 **三选一**，可选「拾取→背包」或「装备→立即生效」：

| 升级 | 效果 | 上限 |
|------|------|:--:|
| 防弹衣 | 生命上限 +20 | 5 |
| 急救包 | 生命恢复 +0.3/秒 | 5 |
| 跑鞋 | 移速 +30 | 5 |
| 穿甲弹 | 伤害 +20% | 5 |
| 快速换弹 | 射速 +15% | 5 |
| 膛线改造 | 子弹速度 +20% | 5 |
| 重型弹药 | 子弹大小 +15% | 5 |
| 双持 | 弹道 +1 | 3 |
| 生存直觉 | 拾取范围 +40 | 5 |
| 狩猎经验 | 经验获取 +20% | 5 |

## 视觉效果

- **屏幕震动**：受伤时摄像机抖动
- **定时冻结**：升级时短暂慢放
- **粒子爆散**：击杀(橙)、经验(绿)、金币(金)、受伤(黄)
- **星空背景**：200 颗星星滚动
- **无敌闪烁**：受伤后 0.5 秒半透明
- **波次进度条**：底部三色条（绿=完/橙=当前/灰=未开）
- **过关面板**：绿色「关卡通过」+ 下一关/返回地图

## 技术细节

- **引擎**：Godot 4.6（GL Compatibility）
- **纹理**：运行时 `Image.create()` 生成像素纹理，无外部素材
- **碰撞层**：玩家=2，敌人=4，子弹碰撞敌人，金币碰撞玩家

### 自动加载 (Autoload)

| 名称 | 作用 |
|------|------|
| `AssetDB` | 像素纹理仓库 |
| `GameManager` | 游戏状态、波次、金币 |
| `UpgradeManager` | 升级池、选择、背包 |
| `ProgressManager` | 关卡进度（5 章 23 关） |
